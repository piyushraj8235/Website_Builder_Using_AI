# 🚀 AI Website Builder SaaS — Production-Ready MERN Application

A production-ready **AI Website Builder SaaS** that can automatically generate and deploy websites in one click.

This project demonstrates a real-world SaaS architecture with authentication, payments, credits, and deployment system. It is designed to simulate how modern AI-based platforms generate websites instantly.

---

## 🔥 Features

* AI Website Generation from user prompt
* One-Click Website Deployment
* Credit-Based Usage System
* Secure Payment Integration using Stripe
* Authentication & Protected Routes
* Smooth UI Animations
* Production Deployment Support

---

## 🛠 Tech Stack

Frontend:

* React.js
* Tailwind CSS
* Framer Motion

Backend:

* Node.js
* Express.js
* MongoDB

Payments:

* Stripe Checkout
* Stripe Webhooks

Deployment:

* Render Cloud Hosting
